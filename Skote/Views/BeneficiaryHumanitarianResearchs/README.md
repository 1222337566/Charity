# Views suggested for Sprint 33

Create the following Razor views using the same style already used in the project:

- `Index.cshtml`
- `Create.cshtml`
- `Details.cshtml`
- `Review.cshtml`
- `Committee.cshtml`

## Minimum blocks in `Details.cshtml`
1. Header:
   - رقم البحث
   - اسم الحالة
   - الحالة الحالية
   - نوع المساعدة
   - تاريخ الطلب / البحث
2. Workflow buttons:
   - إرسال للمراجعة
   - فتح شاشة المراجعة
   - إحالة للجنة
   - قرار اللجنة
3. Main tabs:
   - البيانات الأساسية
   - الأسرة
   - الدخل
   - المصروفات
   - الديون
   - وصف السكن
   - تقرير الباحث
   - مراجعات المراجع
   - قرارات اللجنة
